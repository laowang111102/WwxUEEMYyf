# Python学生成绩管理系统 python590

## 项目概述

本项目是基于Python的学生成绩管理系统，采用Django框架进行开发，前端使用Vue.js，数据库为MySQL。系统功能全面，涵盖了学生、课程、成绩、选课等管理模块。

## 技术栈

- 后端框架：Django
- 数据库：MySQL
- 前端框架：Vue.js
- 开发工具：PyCharm

## 功能模块

- **登录注册**：用户身份验证与注册
- **学生管理**：对学生信息进行增删改查
- **课程管理**：对课程信息进行管理
- **学生成绩管理**：成绩的录入、修改与查询
- **选课信息管理**：学生选课信息管理
- **课程类型管理**：对不同类型的课程进行管理
- **公告管理**：发布公告及管理
- **教师管理**：对教师信息进行管理
- **个人中心**：用户个人信息管理

## 系统角色

- 管理员：负责系统管理与维护
- 教师：负责成绩录入、课程管理
- 学生：查看成绩、选课

## 运行环境

- Python环境
- MySQL数据库
- Node.js环境（前端）

## 部署步骤

1. 配置Python环境
2. 安装依赖库
3. 配置MySQL数据库
4. 运行后端Django服务
5. 构建前端Vue.js项目
6. 部署至Web服务器
